# Resume Skill Analyzer

## Overview
Resume Skill Analyzer is a web application developed as part of the Digital Heroes internship trial task.

It allows users to upload a resume, analyze the skills present, and provide useful insights.

## Features

- Upload Resume
- Skill Analysis
- User-friendly Interface
- Fast Processing

## Tech Stack

- Java
- Spring Boot
- HTML
- CSS
- JavaScript

## How to Run

1. Clone the repository.
2. Open the project in IntelliJ IDEA or Eclipse.
3. Run the Spring Boot application.
4. Open your browser and visit:
   http://localhost:8080
